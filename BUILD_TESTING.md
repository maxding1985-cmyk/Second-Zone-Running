## 安装测试

1. 用 Android 手机打开构建页面或 APK 下载链接。
2. 下载 APK。
3. 如果系统提示“禁止安装未知来源应用”，按提示允许浏览器/文件管理器安装。
4. 安装后打开“跑步聊天”。

## Android Studio 安装后：真机测试准备

### 1. 手机准备

1. 准备一台 Android 真机，电量建议高于 30%。
2. 用手机浏览器打开 APK 下载链接。
3. 下载 APK。
4. 如果系统提示“禁止安装未知来源应用”，进入提示页，允许当前浏览器或文件管理器安装未知来源应用。
5. 安装完成后，桌面上应出现“跑步聊天”。

### 2. 如果要用 USB + Android Studio 辅助排查

这一步不是安装 APK 的必需步骤，但方便抓日志和排查闪退。

1. 手机打开“开发者选项”。
   - 常见路径：设置 -> 关于手机 -> 连续点击“版本号”7 次。
2. 打开“USB 调试”。
   - 常见路径：设置 -> 系统/更多设置 -> 开发者选项 -> USB 调试。
3. 用 USB 线连接手机和电脑。
4. 手机上弹出“是否允许 USB 调试”时，选择允许。
5. 打开 Android Studio。
6. 进入 Device Manager 或右上角设备列表，确认能看到这台手机。

### 3. 用命令确认真机已连接

如果已经完成 USB 调试，可以在终端执行：

```bash
export PATH="/usr/local/share/android-commandlinetools/platform-tools:$PATH"
adb devices
```

预期看到类似：

```text
List of devices attached
R58Nxxxxxxx    device
```

如果显示 `unauthorized`，看手机屏幕，点击允许 USB 调试。

### 4. 用 adb 安装 APK（可选）

如果手机直接下载 APK 安装失败，可以用电脑下载 APK 后通过 adb 安装。

```bash
mkdir -p /Users/user/Documents/跑步聊天/artifacts
curl -L "https://expo.dev/artifacts/eas/Rsx7TcVN8JXl2KChBu2ew6zi7O480XhxoE3EP0_94J0.apk" \
  -o /Users/user/Documents/跑步聊天/artifacts/run-chat-preview.apk

export PATH="/usr/local/share/android-commandlinetools/platform-tools:$PATH"
adb install -r /Users/user/Documents/跑步聊天/artifacts/run-chat-preview.apk
adb shell monkey -p com.runchat.mvp 1
```

预期结果：

- `adb install` 输出 `Success`。
- 手机自动打开“跑步聊天”。

## 当前包的限制

当前 `preview` 包的 API 地址是：

```text
http://127.0.0.1:4000
```

这个地址在真机上指向手机自己，不是开发电脑。所以真机安装后大概率会显示“离线体验中”。这个包适合先测 App UI、跑步流程、跑后报告的本地体验。

如果要测试完整后端数据闭环，需要先部署 API 到公网，然后把 `apps/mobile/eas.json` 里的 `EXPO_PUBLIC_API_URL` 改成公网地址后重新出包。

### 建议测试顺序

第一轮只需要 15-20 分钟，建议按这个顺序跑：

1. `TC-001 安装与首次启动`
2. `TC-003 正常完成一次聊天跑`
3. `TC-004 跑后报告与反馈表单`
4. `TC-006 暂停与继续`
5. `TC-007 安全与隐私页`

如果第一轮没有明显问题，再补：

1. `TC-002 首页辅助入口`
2. `TC-005 提前结束路径`
3. `TC-008 兼容性冒烟`

### 测试范围说明

当前 APK 是 `preview` 包，API 地址为 `http://127.0.0.1:4000`。在真机上这通常无法连接后端，因此本轮重点验证：

- App 能安装、启动、导航。
- 跑步前安全提示完整可见。
- 跑中 AI 陪跑流程可用。
- 离线状态下仍能完成跑步和跑后报告。
- 跑后反馈表单在离线时有合理提示。

本轮不验证：

- 后端真实写入。
- 运营看板真实数据增长。
- JSON/CSV 导出。

这些需要公网 API 版本的 APK。
