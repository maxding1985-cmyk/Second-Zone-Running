## 3. 通用判定标准

### 3.0 正确输出速查

| 场景 | 正确输出 |
| --- | --- |
| 首页启动 | 显示 `AI 二区语音陪跑`、`只要还能聊天，就别急着加速。`、`开始聊天跑` |
| 跑前页 | 显示 `今天想怎么跑？`、`跑步场景`、`AI 陪跑风格`、`下一步` |
| 安全页 | 显示 `先确认一件事`、`运动安全提示`、`我知道了，开始跑` |
| 跑中页 | 显示计时器、`AI 陪跑员`、`轻松`、`有点喘`、`太累了`、`安静一会儿`、`结束并生成报告` |
| 当前真机包连接状态 | 通常显示 `离线体验中`，这是正确输出 |
| 正常结束报告页 | 显示 `轻松完成`、`这次你不是一个人跑。`、`陪跑时长`、`跑中反馈`、`及时慢下来`、`AI 总结` |
| 提前结束报告页 | 显示 `已记录这次尝试` 和对应 `提前结束原因` |
| 离线反馈提交 | 显示 `当前未连接服务端，反馈稍后再提交。`，这是正确输出 |
| 运营看板离线 | 显示无法连接 API 的提示，这是正确输出 |

### 3.1 Pass 标准

用例同时满足以下条件，判定为 Pass：

1. 实际结果与“正确输出 / 预期结果”一致。
2. 操作过程中没有闪退、白屏、卡死。
3. 页面主要文案可见，按钮可点击。
4. 当前版本已说明的离线提示不算失败。

### 3.2 Fail 标准

出现任一情况，判定为 Fail：

1. App 无法安装或无法启动。
2. 页面白屏、黑屏、闪退、卡死超过 10 秒。
3. 关键按钮不可点击，无法继续主流程。
4. 报告页无法生成。
5. 安全提示缺失或主要文案严重截断。

### 3.3 当前版本的已知预期现象

由于当前 APK 的 API 地址是 `http://127.0.0.1:4000`：

- 跑中页显示 `离线体验中` 是正确输出。
- 运营看板显示无法连接 API 是正确输出。
- 跑后反馈提交失败并显示离线提示是正确输出。
- 不要求后端指标增长。

## TC-A02 adb 安装 APK

| 字段 | 内容 |
| --- | --- |
| 用例 ID | TC-A02 |
| 用例名称 | adb 安装 APK |
| 优先级 | P1 |
| 测试目标 | 验证通过电脑 adb 可安装并启动 APK |
| 前置条件 | Android Studio 已安装；手机已开启 USB 调试；手机通过 USB 连接电脑 |
| 测试输入 | 本地 APK 文件：`artifacts/run-chat-preview.apk` |

### 准备命令

```bash
cd /Users/user/Documents/跑步聊天
mkdir -p artifacts
curl -L "https://expo.dev/artifacts/eas/Rsx7TcVN8JXl2KChBu2ew6zi7O480XhxoE3EP0_94J0.apk" \
  -o artifacts/run-chat-preview.apk
export PATH="/usr/local/share/android-commandlinetools/platform-tools:$PATH"
adb devices
```

### 操作步骤与预期输出

| 步骤 | 操作 | 测试输入 | 正确输出 / 预期结果 |
| --- | --- | --- | --- |
| 1 | 执行 `adb devices` | 手机 USB 连接 | 输出中存在一行设备，状态为 `device` |
| 2 | 如果状态为 `unauthorized` | 手机屏幕弹窗 | 手机上点击“允许 USB 调试”，再次执行 `adb devices` 后状态变为 `device` |
| 3 | 执行安装命令 | `adb install -r artifacts/run-chat-preview.apk` | 终端输出 `Success` |
| 4 | 执行启动命令 | `adb shell monkey -p com.runchat.mvp 1` | 手机自动打开“跑步聊天” |

### 最终正确结果

终端输出应包含：

```text
Success
```

手机应进入 App 首页，看到：

```text
只要还能聊天，就别急着加速。

```

## 严重程度定义

| 严重程度 | 定义 | 示例 |
| --- | --- | --- |
| Blocker | 阻塞测试继续 | APK 无法安装、App 打不开 |
| Critical | 核心链路不可用 | 无法开始跑、无法进入报告页 |
| Major | 重要功能异常但可绕过 | 反馈按钮无反应、暂停异常 |
| Minor | 体验问题 | 文案截断、样式错位 |
| Trivial | 轻微问题 | 个别标点、间距不一致 |
